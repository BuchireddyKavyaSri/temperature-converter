# Temperature Converter Web Application

## Description
This project is a simple Temperature Converter web application developed using HTML, CSS, and JavaScript. It allows users to convert temperature values between Celsius, Fahrenheit, and Kelvin using an easy and interactive interface.

## Features
- Convert Celsius to Fahrenheit
- Convert Fahrenheit to Celsius
- Convert Celsius to Kelvin
- Convert Kelvin to Celsius
- Simple and user-friendly interface
- Instant temperature conversion

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Structure
temperature-converter
│
├── index.html
├── style.css
└── script.js

## How to Run the Project
1. Download or clone the repository.
2. Open the project folder.
3. Double-click the **index.html** file.
4. The application will open in your browser.
